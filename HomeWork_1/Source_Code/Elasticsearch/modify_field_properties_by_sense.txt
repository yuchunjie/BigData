[注意]：修改field 的property，需要先將elasticsearch 整個index給刪除

[Step 1]
DELETE /spammail

[Step 1.1]
GET /spammail/_mapping/6attr

[Step 2]
POST /spammail

            "properties": {

                  "type": "string",

               "file_length": {

               "file_name": {
                  "type": "string",
                  "index" : "not_analyzed"

               "hyperlink": {

                  "type": "string"

                  "type": "long"

                  "type": "string"
